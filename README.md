# Shopping Cart System

A simple console-based Shopping Cart System built using Python and MySQL.

## Features

- Add products to the cart
- Remove products from the cart
- View all items in the cart
- Generate the final bill
- Simulate payment
- Store data using MySQL

## Technologies Used

- Python
- MySQL
- mysql-connector-python

## Files

- **big one.py** – Main program
- **test1.py** – Creates and fills the Shop table
- **test2.py** – Creates the Cart table

## How to Run

1. Create a MySQL database named `ShoppingCart`.
2. Run `test1.py`.
3. Run `test2.py`.
4. Run `big one.py`.
